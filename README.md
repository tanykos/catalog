# Тестовое задание: Frontend Developer (React/Next.js)

## Описание проекта
Каталог товаров с фильтрацией, сортировкой и возможностью добавления комментариев.

**Технологии:** Next.js 13, TypeScript, Redux Toolkit, SCSS Modules

## Задание
В проекте реализована базовая структура и верстка. Необходимо реализовать недостающий функционал.

### Задача 1: Получение и отображение данных
Реализуйте загрузку данных карточек и их отображение на странице каталога.

### Задача 2: Фильтрация по странам
Реализуйте фильтр карточек по странам (поле country).

### Задача 3: Счётчик товаров
Отобразите количество товаров, соответствующих текущим фильтрам.

### Задача 4: Добавление в избранное (не обязательно)
Реализуйте функционал добавления и удаления товара из избранного.

## Критерии оценки
- Корректность работы функционала
- Работа с Redux Toolkit
- Оптимизация производительности
- Качество реализации

## Запуск проекта
```bash
npm install
npm run dev
```
